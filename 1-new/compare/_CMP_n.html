Compares two big endian arrays, |a| = |b|

Parameters:

  • a must be an Array.
    (first operand)

  • ai must be a Number.
    (a left)

  • aj must be a Number.
    (a right)

  • b must be an Array.
    (second operand)

  • bi must be a Number.
    (b left)

Returns a Number
(1 if a > b; 0 if a = b; -1 otherwise.)

const _CMP_n = function ( a , ai , aj , b , bi ) { for (; ai < aj; ++ai, ++bi) { if (a[ai] > b[bi]) return 1; if (a[ai] < b[bi]) return -1; } return 0; }; exports._CMP_n = _CMP_n;
